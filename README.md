# Investor Risk and Return Analysis (Power BI)

​## Project Overview 

​This project provides a comprehensive analysis of investor behavior, risk appetite, and return expectations. By transforming qualitative survey data into quantitative metrics, the dashboard reveals how demographics like age and gender influence financial decision-making and investment choices.

​ ## Tools Used

​* **Power BI Desktop:**  Dashboard design and data visualization.
​* **Power Query (M):** Data cleaning, splitting column ranges, and normalization.
​* **DAX:** Calculated measures for average return expectations and sample counts.
​* **GitHub:** Version control and portfolio hosting.

​## Key Insights

​* **Risk vs. Age:** Investor return expectations show significant volatility across age groups, with younger investors generally targeting higher-risk avenues for capital appreciation.
​* **Preferred Avenues:** Mutual Funds emerged as the dominant investment choice across the sample, followed by Equity and Fixed Deposits.
​* **Objective-Driven Investing:** "Capital Appreciation" is the primary goal for the majority of participants, directly correlating with a preference for Mutual Funds and Equity Market instruments.
​* **Gender Perspectives:** The analysis highlights distinct patterns in how male and female investors approach long-term vs. short-term investment durations.

​## Data Transformation Steps

​* **Data Normalization:** Converted text-based return ranges (e.g., "20%-30%") into numeric averages (25.0%) using Power Query.
​* **Custom Sorting:** Implemented a conditional sort column for "Duration" to ensure logical chronological ordering (Less than 1 year \rightarrow 1-3 years \rightarrow etc.).
​* **Data Cleaning:** Handled null values and standardized naming conventions for investment objectives.
